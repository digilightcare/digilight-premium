# Logo Files

Please replace `digilight-logo.png` with the actual Digilight logo file you provided.

The logo should be:
- Transparent background PNG
- Approximately 418x66 pixels (or similar aspect ratio)
- Shows "DIGILIGHT" with blue and orange branding

You can use any of these files you uploaded:
1. Digilight_Word_Logo-Transparent 418x66.png (recommended for header)
2. Digilight Logo Transparent.png (for favicon/icon)
3. Digilight Logo- Final.png (for social media)

Simply rename your chosen file to `digilight-logo.png` and place it in this images folder.
