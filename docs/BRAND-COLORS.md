# Digilight India - Official Brand Colors

## Primary Brand Colors

### Blue (Primary)
- **Hex Code:** `#0056d2`
- **CSS Variable:** `--primary-blue`
- **Usage:** 
  - Logo primary color
  - Section headings
  - Primary buttons
  - Links and hover states
  - Icons and accents
  - Gradients

### Orange (Accent)
- **Hex Code:** `#ff6a00`
- **CSS Variable:** `--accent-orange`
- **Usage:**
  - Logo accent color
  - Call-to-action buttons
  - Highlights and emphasis
  - Section borders (top border on cards)
  - Gradients
  - Interactive elements

## Implementation

These colors are defined in `styles.css` as CSS variables:

```css
:root {
    --primary-blue: #0056d2;
    --accent-orange: #ff6a00;
}
```

All instances of these colors across the website have been standardized:
