## üîß Required Actions (Insert Your IDs)

### Step 1: Google Analytics 4 (GA4)
1. Go to https://analytics.google.com
2. Create a GA4 property for your website
3. Get your Measurement ID (format: G-XXXXXXXXXX)
4. **Replace in all HTML files:**
   - Find: `YOUR_GA4_MEASUREMENT_ID`
   - Replace with: Your actual GA4 Measurement ID

### Step 2: Google Tag Manager (GTM)
1. Go to https://tagmanager.google.com
2. Create a container for your website
3. Get your GTM ID (format: GTM-XXXXXXX)
4. **Replace in all HTML files:**
   - Find: `YOUR_GTM_ID`
   - Replace with: Your actual GTM ID

### Step 3: Google Search Console
1. Go to https://search.google.com/search-console
2. Add your property: https://digilightindia.netlify.app
3. Choose "HTML tag" verification method
4. Copy the verification code
5. **Replace in all HTML files:**
   - Find: `YOUR_GOOGLE_SEARCH_CONSOLE_VERIFICATION_CODE`
   - Replace with: Your verification code

### Step 4: Submit Sitemap
1. After verifying Search Console
2. Go to Sitemaps section
3. Submit: `https://digilightindia.netlify.app/sitemap.xml`
